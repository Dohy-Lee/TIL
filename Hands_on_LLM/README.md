# 핸즈온 LLM

학습 날짜: 2026.05.23 ~ 2026.

RAG, 프롬프트 엔지니어링, 에이전트를 이해하기 위해 『핸즈온 LLM』을 학습하며 정리한 저장소

RAG, 프롬프트 엔지니어링, 에이전트 등 LLM 활용의 핵심 주제를 직접 정리하고 실습하며 이해하는 것을 목표로 합

책의 전체 순서를 그대로 따라가기보다는, 현재 관심 있는 주제인 **RAG**, **프롬프트 엔지니어링**, **에이전트** 챕터를 우선적으로 학습함
각 챕터를 따라가며 주요 개념을 정리하고, 직접 코드를 실행하며 헷갈렸던 부분을 기록

## 학습 목표

- LLM의 기본 동작 방식과 활용 구조 이해
- RAG의 전체 흐름 이해
  - 문서 수집
  - 청킹
  - 임베딩
  - 벡터 검색
  - 검색 결과 기반 답변 생성
- 프롬프트 엔지니어링의 주요 기법 정리
- 에이전트 구조와 Tool 사용 방식 이해
- 책의 예제 코드를 직접 구현하며 실습 경험 축적
- 이후 프로젝트를 위한 RAG, 프롬프트, 에이전트 기반 다지기

## 정리 내용

- 각 챕터의 핵심 개념 요약
- 책 예제 기반 실습 코드
- 헷갈렸던 개념에 대한 개인 정리
